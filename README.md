## ☁️ What I Do

**Cloud Infrastructure & IaC**
Provision and maintain production-grade environments on Azure, AWS, and Hetzner
Cloud using Terraform and Terragrunt — multi-environment (dev/staging/prod) from
a single codebase, remote state, DynamoDB locking, and security gates in CI before
anything is applied.

**GitOps & Kubernetes**
Build and manage Kubernetes platforms (AKS, k3s HA) with ArgoCD, Helm, and KEDA
autoscaling. One git push deploys everything — drift detected and self-healed
automatically. No manual kubectl. No manual helm install. Just git.

**CI/CD Pipelines**
Design end-to-end pipelines using GitHub Actions, Azure Pipelines, GitLab CI/CD,
and Jenkins — build, test, security scan, deploy, and health check. Every stage
blocks on failure, nothing slips through.

**Security**
Harden infrastructure at every layer — Trivy CVE scanning (reduced CVEs from 23
to 0), TruffleHog secrets scanning on every PR, OPA/Gatekeeper registry
constraints, tfsec CI gates, Pod Security Standards, and Azure Key Vault secrets
integration via SecretProviderClass.

**Observability**
Deploy full three-pillar observability stacks — Prometheus, Grafana, Loki, Tempo,
Alertmanager, and OpenTelemetry. Metrics, logs, and traces fully correlated by
trace_id. Custom alert rules covering pod crashes, high memory, and node health.
